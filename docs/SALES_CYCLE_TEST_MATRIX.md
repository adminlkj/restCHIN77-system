| M5.04 | تقرير المبيعات اليومي | CASH += 115 | JE (1111 debit) | ⏳ |

## خريطة طريق إكمال الاعتماد

1. **إقلاع بيئة اختبار** وفق `RUNBOOK.md` (قسم 3).
2. **تشغيل seed script** لإنشاء بيانات تجريبية.
3. **تنفيذ كل Test Case** يدوياً أو بسكربت آلي.
4. **تعبئة الحقول** (PASS/FAIL) + لقطة شاشة/دليل.
5. **تجميع النتائج** في شهادة الاعتماد.
6. **معالجة أي FAIL** ثم إعادة M1-M8 (regression).

عند اكتمال كل Test Cases بـ PASS، تُمنح دورة البيع شهادة `CERTIFIED`.
